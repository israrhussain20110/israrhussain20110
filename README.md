### 🚀 Featured Projects

#### 🔍 AI Incident Root Cause Investigator
Analyzes logs, Git commits, and monitoring alerts to surface the most probable root cause of production incidents using semantic search and LLM reasoning.
* **Features:** Semantic search across logs/alerts, structured LLM reasoning, automatic timeline reconstruction.
* **Stack:** `FastAPI` `LangChain` `Vector Search` `Python`

#### 📚 Document Assistant Chatbot
Production RAG pipeline for querying large document collections at scale.
* **Features:** Full document ingestion/chunking, 35% faster responses via streaming, handles 100+ concurrent users.
* **Stack:** `FAISS` `HuggingFace` `FastAPI`

#### 📈 AI Inventory Forecast System
Fully automated ML pipeline that predicts inventory demand from historical operational data.
* **Features:** Time-series demand forecasting, 50% reduction in manual reporting effort.
* **Stack:** `PyTorch` `Python`

#### 📝 Custom LLM Summarizer
Handles documents that exceed a single context window for coherent, accurate summaries.
* **Features:** Hierarchical document chunking, multi-stage context-aware summarization for arbitrarily long inputs.
* **Stack:** `HuggingFace` `Python`

#### 🏥 Pharmacy Analytics Service
Backend analytics platform that auto-generates KPI reports and operational insights from data pipelines.
* **Features:** Automated KPI aggregation, clean reporting APIs, 50% reduction in manual work.
* **Stack:** `FastAPI` `MongoDB`

#### 🦴 Knee Osteoporosis Classifier
Medical image classification model for automated diagnostic pre-screening, reducing radiologist workload.
* **Features:** 85% validation accuracy, 40% reduction in clinical evaluation time.
* **Stack:** `PyTorch` `VGG16` `CNN`
